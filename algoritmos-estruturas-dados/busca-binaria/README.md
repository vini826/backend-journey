# Busca Binária

A busca binária é um algoritmo eficiente para procurar um item em uma lista **ordenada**. Ela divide a lista ao meio, descartando a metade que não contém o item procurado.

## Implementações

- [JavaScript](busca_binaria.js)
- [Python](busca_binaria.py)

## Complexidade de Tempo

- Melhor caso: O(1) (Quando o item procurado está no meio da lista)
- Pior caso: O(log n)
- Caso médio: O(log n)
